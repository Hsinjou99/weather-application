# Weather Application

A simple web application to get current weather information using WeatherAPI.

## Features

- Enter a city name to get current weather data
- Displays temperature, weather condition, humidity, and wind information

## Setup

1. Get an API key from https://www.weatherapi.com
2. In script.js, replace YOUR_API_KEY with the API key you just got
3. Open index.html in a web browser

## Files

- index.html: Main HTML structure
- styles.css: CSS styling
- script.js: JavaScript for getting and showing weather info
